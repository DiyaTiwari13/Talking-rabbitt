# Talking Rabbitt MVP

Talking Rabbitt is a conversational analytics tool that allows users to interact with business data using natural language.

## Features

- Upload CSV datasets
- Ask questions about the data
- Get instant answers
- Automatic data visualization

## Technology Stack

Python  
Streamlit  
Pandas  
Matplotlib  

## How to Run the Project

1. Install dependencies

pip install -r requirements.txt

2. Run the application

streamlit run app.py

3. Open in browser

http://localhost:8501
